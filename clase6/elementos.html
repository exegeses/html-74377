Tipos de elementos de HTML según el CSS

Elementos de línea

Los elementos de línea se presentan uno al lado del otro (inline)
Ocupan su propio espacio y no aceptan ancho ni altura

Elementos de bloque

Los elementos del bloque ocupan todo el ancho del contenedor. Se ubican uno debajo del otro.
Aceptan ancho y altura

Elementos de línea-bloque

Los elementos de líneas lo que son principalmente las imágenes. Éstas se presentan en línea (una al lado de la otra), pero además pueden aceptar ancho y altura.
